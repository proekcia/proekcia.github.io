# Landing Page Project

## Описание
Верстка лендинга по дизайну из Фигмы с адаптивом под планшет и мобилку.

## Как работаем
- Верстаем **по блокам**: пользователь скидывает скриншот блока + CSS из Фигмы
- Адаптив делаем **резиновым в формате VW** (fluid layout)
- Шрифты подключаем в начале работы
- В новых сессиях — читай этот файл чтобы восстановить контекст

## Структура файлов
- `index.html` — основной HTML файл
- `style.css` — основные стили
- `fonts/` — папка со шрифтами

## Брейкпоинты
- Desktop: 1440px (базовая ширина дизайна)
- Mobile: 375px
- Медиазапрос для мобилки: @media (max-width: 768px)

## Шрифты
- Inter Regular (400) — `fonts/Inter-Regular.otf`
- Inter Bold (700) — `fonts/Inter-Bold.otf`
